The Honor Chain

The DPYC Honor Chain is a voluntary community organized as a Network Society. Members agree to use Bitcoin for commerce, avoid storing customer PII, identify their upstream Authority, and accept community governance.

Membership tiers: Citizen (consumer) → Advocate (utility service) → Operator (runs an MCP service) → Authority (certifies operators) → First Curator (root of trust).

First Curator (Prime Authority) │ ├── Authority A │ ├── Operator: TheBrain MCP │ ├── Operator: Schwab MCP │ ├── Operator: eXcalibur MCP │ └── Operator: TaxSort MCP │ ├── Authority B │ ├── Operator B1 │ └── Operator B2 │ └── Advocate: OAuth2 Collector
