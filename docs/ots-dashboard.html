Open Timestamps notarization status across the Tollbooth operator network

How this works: Each Tollbooth operator periodically snapshots all patron balances into a SHA-256 Merkle tree and submits the root hash to OpenTimestamps calendar servers. Those calendars anchor the hash into a Bitcoin block, creating an immutable proof that the ledger state existed at that moment. This page shows the notarization activity across the DPYC Honor Chain.
